# Vibration_sensor.ino

##




## 使い方



#### メインコマンド
| コマンド | 内容                               |
|----------|------------------------------------|
| START    | 計測開始                           |
| RESET    | マイコンのリセット                 |
| GET      | パラメータやバージョン情報等を取得 |
| SET      | パラメータ等を設定                 |	

#### GET コマンドオプション

| GETコマンド | 内容                       |
|-------------|----------------------------|
| VERSION     | バージョン情報             |
| ALGOLISM    | 振動判定アルゴリズムの番号 |
| THRESH      | 振動判定時の閾値           |
| INTERVAL    | 振動判定の周期(msec)       |
| DOUT_PIN    | 振動判定結果出力のピン番号 |
| AIN_PIN     | 振動センサ入力ピン番号     |
| COMM_TYPE   |                            |
| BPS         | 通信ボーレート             |

#### SET コマンドオプション

| SETコマンド | 内容                       |
|-------------|----------------------------|
| ALGOLISM    | 振動判定アルゴリズムの番号 |
| THRESH      | 振動判定時の閾値           |
| INTERVAL    | 振動判定の周期(msec)       |
| DOUT_PIN    | 振動判定結果出力のピン番号 |
| AIN_PIN     | 振動センサ入力ピン番号     |
| COMM_TYPE   |                            |

#### ALGOLISM 番号

| ALGO TYPE | VALUE                                               |
|-----------|-----------------------------------------------------|
| 1         | 前回測定(メインループでの) 差分のINTERVAL間の平均値 |
| 2         | INTERVAL間のMAX、MINの差分                          |
| 3         | INTERVAL間平均値                                    |
| 4         | 直値(平均なし)                                      |
| 5         | INTERVAL間の分散                                    |
